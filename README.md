# hcredit-app
## Запуск

```cli
# Установите зависимости
npm install

# Создать файл .env.local в корневом каталоге добавить строку
NEXT_PUBLIC_HOSTNAME={доменное имя с протоколом}
NEXT_PUBLIC_DIR={директория в которой храниться все файлы}

# Запустите сервер для локальной разработки
npm run dev

# Залогиньтесь в HCM по адресу
http://localhost:3000/view_doc.html?mode=default

# Вернитесь в приложение http://localhost:3000/

# После всех изменение запустить сборку
npm run build

Все файлы появяться в директории ./out/ эти файлы закинуть 
в директорию где будет храниться текущей проект
```
